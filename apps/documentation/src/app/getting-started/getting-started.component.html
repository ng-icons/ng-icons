Importing Icons

Only include the icons you want to use 

import { NgIconsModule } from '@ng-icons/core';
import { featherAirplay } from '@ng-icons/feather-icons';
import { heroUsers } from '@ng-icons/heroicons/outline';

@NgModule({
  imports: [
    BrowserModule,
    NgIconsModule.withIcons({ featherAirplay, heroUsers }),
  ],
})
export class AppModule {}

import { NgIconComponent, provideIcons } from '@ng-icons/core';
import { featherAirplay } from '@ng-icons/feather-icons';
import { heroUsers } from '@ng-icons/heroicons/outline';

@Component({
    selector: 'app-root',
    templateUrl: './app.component.html',
    imports: [NgIconComponent],
    viewProviders: [provideIcons({ featherAirplay, heroUsers })]
})
export class AppComponent {}

Using Icons

Use the ng-icon component to display the icon. 

<ng-icon name="heroUsers"></ng-icon>

Dynamic Icons

Icons can be loaded dynamically using the provideNgIconLoader function.
Define a function that returns and observable or a promise that resolves to the svg string. 

provideNgIconLoader(name => {
  const http = inject(HttpClient);
  return http.get(`/assets/icons/${name}.svg`);
})
